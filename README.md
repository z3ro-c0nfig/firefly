1 - [**Download firefly here**](https://github.com/z3ro-c0nfig/firefly/releases)

2 - Run **"firefly-v2.exe"**

3 - Enter your **bot token**.

4 - **Invite** the bot to a server.

5 - Use the command **".nuke"** to **nuke** the server.
